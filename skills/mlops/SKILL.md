# MLOps

Implement and manage ML operations including pipelines, deployment, and monitoring.

## When to Use

- Setting up ML pipelines
- Deploying ML models to production
- Monitoring model performance
- ML infrastructure management

## Capabilities

- Build and orchestrate ML pipelines (Airflow, Kubeflow)
- Deploy models to production (Docker, Kubernetes, Sagemaker)
- Set up model monitoring and drift detection
- Implement CI/CD for ML systems
- Manage feature stores and data pipelines
- Optimize model performance and inference
- Track experiments and versions
