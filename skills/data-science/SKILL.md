# Data Science

Perform data analysis, create visualizations, and build machine learning models.

## When to Use

- User needs to analyze data or create visualizations
- Building predictive models
- Exploratory data analysis
- Statistical analysis

## Capabilities

- Load and clean datasets (CSV, JSON, databases)
- Create visualizations (matplotlib, seaborn, plotly)
- Perform statistical analysis
- Build and evaluate ML models
- Create data pipelines
- Generate reports and insights
