# Smart Home

Manage and automate smart home devices and IoT systems.

## When to Use

- User has smart home devices to configure
- Setting up home automation routines
- Troubleshooting IoT connectivity issues

## Capabilities

- Configure smart home hubs (Home Assistant, etc.)
- Set up automation routines and triggers
- Manage IoT devices and protocols (Zigbee, Z-Wave, MQTT)
- Monitor device status and energy usage
- Create voice commands and shortcuts
- Troubleshoot connectivity issues
