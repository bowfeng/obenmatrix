# Media

Process and manipulate media files including images, video, and audio.

## When to Use

- User needs to convert or edit media files
- Processing images, video, or audio
- Working with multimedia formats

## Capabilities

- Convert between media formats (FFmpeg, ImageMagick)
- Batch process image and video files
- Extract metadata from media files
- Generate thumbnails and previews
- Apply filters and effects
- Compress media for web or mobile
- Create GIFs from video or images
