# DevOps

Help with continuous integration, deployment, containerization, and infrastructure management.

## When to Use

- User mentions Docker, Kubernetes, CI/CD, or infrastructure
- Need to set up deployment pipelines
- Working with cloud providers or container orchestration

## Capabilities

- Write and debug Dockerfiles and docker-compose files
- Configure CI/CD pipelines (GitHub Actions, GitLab CI, etc.)
- Manage Kubernetes deployments and services
- Create infrastructure as code (Terraform, CloudFormation)
- Set up monitoring and logging
- Optimize build and deployment processes
