# Diagramming

Create visual diagrams, flowcharts, and documentation visuals.

## When to Use

- Need to visualize system architecture or processes
- Creating flowcharts or sequence diagrams
- Documenting workflows or systems

## Capabilities

- Create Mermaid diagrams (flowcharts, sequences, Gantt)
- Generate PlantUML diagrams
- Create architecture diagrams
- Design process flows and workflows
- Build organizational charts
- Create entity-relationship diagrams
- Visualize data and relationships
