# GIFs

Create, edit, and optimize GIF animations for various use cases.

## When to Use

- Creating GIFs from video or images
- Optimizing GIF file sizes
- Editing GIF frames and colors
- Adding text or effects to GIFs

## Capabilities

- Convert video to GIF (FFmpeg)
- Create GIFs from image sequences
- Optimize GIF colors and file size
- Add text overlays and effects
- Create animated GIFs from scratch
- Extract frames from existing GIFs
- Batch process multiple GIFs
