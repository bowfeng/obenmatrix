# Dogfood

Perform systematic quality assurance testing on applications.

## When to Use

- Testing new features or products
- Finding bugs and usability issues
- Creating bug reports with evidence
- QA testing workflows

## Capabilities

- Systematic UI/UX testing
- Cross-browser compatibility testing
- Performance testing
- Accessibility testing
- Create detailed bug reports
- Capture screenshots and logs as evidence
- Track and prioritize issues
