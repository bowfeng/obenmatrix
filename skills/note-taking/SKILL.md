# Note Taking

Organize information, create knowledge bases, and manage documentation.

## When to Use

- Creating or organizing notes
- Building knowledge bases
- Writing technical documentation
- Managing personal or team knowledge

## Capabilities

- Create and structure markdown notes
- Build personal knowledge bases
- Write technical documentation
- Organize information with tags and categories
- Create note templates
- Search and retrieve stored information
- Link related notes and concepts
